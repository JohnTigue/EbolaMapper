About

Purpose

This web-app provides a high level dashboard for epidemological disease outbreaks. This project has two deliverables: define a data standard for outbreak information and produce software which reads the data standard.

Outbreak data standard

The first deliverable of the project is to define a data standard for epidemiological outbreaks. The data standard (or API, or format) is called Outbreak Time Series Specification.

Outbreak software

Omolumeter is the second of the two deliverables: software which reads Outbreak Time Series Specification compliant data and then visualizes that data. The OTSS files are read and a JavaScript object model of the data is made available to the visualizations to render. Data is read from multiple sources (web sites), a task with which the standard Outbreak API is most useful. The Omolumeter is the software which visualizes the Outbreak Time Series data.
