About

Purpose

The subject of this project is epidemiological disease outbreaks. This Omolumeter app provides a high level dashboard for outbreaks, with information presented as an explorable explanation.

Deliverable #1: outbreak data standard

The first deliverable of the project is to define a data standard for epidemiological outbreaks. The data standard is called Outbreak Time Series Specification.

Deliverable #2: outbreak visualization software

Omolumeter is the second of the two deliverables: software which reads Outbreak Time Series Specification compliant data and then visualizes that data. Data is read from multiple API sources (web sites). The Omolumeter proves that arbitrary web apps can be built which work with the new data standard. This sofware runs completely in the web browser; there is no server-side code. On the web server there is nothing but static files. Omolumeter can also be packages as native apps for iOS and Andriod (via Cordova).

Version
